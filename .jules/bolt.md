## 2026-04-23 - Removed Duplicate DOM Initialization in `index.html`
**Learning:** The frontend contained duplicate logic for DOM selection and event listener binding inside the same `<script>` block for `index.html`. It additionally contained an incorrect ID `feedbackForm` instead of `upgradeForm` referencing the main form logic inside the `DOMContentLoaded` listener block. This resulted in an initial JavaScript error `Cannot read properties of null (reading 'addEventListener')` and unneeded additional event listeners.
**Action:** Next time, make sure to check for duplicate/redundant JavaScript snippets and clean up incorrect IDs references. Eliminating duplicate bindings and incorrect references helps improve maintainability and removes the overhead of registering extra listeners.
